# テニスの王子様 年齢比較ダッシュボード 要件定義書

## 1. プロジェクトの目的
ユーザーが「自分の年齢」を入力することで、「テニスの王子様」に登場する中学生キャラクターの年齢・身長・体重等と比較し、自分とキャラクターの位置関係を視覚的に楽しめるダッシュボードを提供する。

## 2. ターゲットユーザー
- テニスの王子様のファンで、自分とキャラクターの設定上の年齢を比較してみたい人
- （中学生でこれだけ大人びているのか、というギャップを楽しみたい人）

## 3. 主要機能
### 3.1. ユーザー入力機能
- **年齢入力**: 1桁〜3桁の整数。初期値は空白。入力によってダッシュボード全体がリアクティブに変化。
- **身長/体重入力（オプション拡張として想定）**: 自分とキャラクターの体格差も比較できるようにする。

### 3.2. 比較・ダッシュボード表示機能
- **サマリー指標 (KPI)**
  - あなたと同い年のキャラクター数
  - あなたより年上のキャラクター数 / 年下のキャラクター数
  - 全キャラクターの中でのあなたの年齢のポジション
- **グラフ表示**
  - **年齢分布比較**: 年齢ごとのキャラクター数分布の中に、ユーザーの年齢をハイライト表示（棒グラフ/ヒストグラム）。
  - **学校別比較**: 各学校における平均年齢・平均身長などの全体傾向とユーザーの比較。
- **キャラクターデータ一覧表**
  - 一覧表として全キャラクターを表示し、ユーザーとの「年齢差」カラムを動的に追加・表示する。
  - 列構成: キャラクター名, 学校, 学年, 8/23時点年齢, 年齢差, 身長, 体重, 血液型, 利き腕, 家族

### 3.3. フィルター機能（ダッシュボードガイドライン準拠）
画面上部もしくは左部に以下のフィルターを配置する。
1. **学校フィルター (マルチセレクト)**: 青春学園、氷帝学園、立海大附属など。
2. **学年フィルター (マルチセレクト)**: 1年, 2年, 3年。
3. **年齢差フィルター (ラジオ/セレクト)**: 「同い年」「年下」「年上」「全件」
4. **利き腕 / 血液型フィルター (オプション)**

## 4. UI/UX・デザイン制約（デジタル庁ダッシュボードガイドライン準拠）
- **左上から右下への視線誘導**: 左上に年齢入力とサマリー指標を置き、詳細な表やグラフを下・右に配置。
- **カラーパレットの統一**: 学校ごとのテーマカラー（青学＝青、氷帝＝水色/グレー、立海＝黄色など）を取り入れつつ、原色を避けたモダンな色合いにする。
- **アクセシビリティ**: 適切なコントラスト比を確保し、フィルターや入力項目の操作をシンプルに保つ。

## 5. 技術スタック・インフラ
- **言語/フレームワーク**: Next.js (App Router), React, TypeScript
- **スタイリング**: Tailwind CSS （モダンで美しいダッシュボード構築のため許可を前提としつつ、vanilla CSSも可）
  - ※デザインの複雑さを考慮し、通常はTailwind CSS, shadcn/uiなどのモダンツールが適しています（ここはユーザーと確認）
- **グラフ描画**: Recharts または Chart.js
- **デプロイ環境**: Vercel
